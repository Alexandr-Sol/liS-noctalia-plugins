# Agy Panel

[English](README.md) | [Русский](README.ru.md)

---

Плагин для Noctalia Shell, интегрирующий интерфейс Antigravity CLI (через встроенный скрипт-мост `agy-bridge`) в боковую панель чата.

---

## Зависимости и требования

* `antigravity-cli` (предоставляет команду `agy` CLI)
* `wl-clipboard` (предоставляет `wl-paste` и `wl-copy` для работы с буфером обмена в Wayland)
* `python` (для запуска скрипта-моста)

## Возможности

- Чат с прокруткой и потоковым выводом ответов (без визуальных артефактов при прокрутке)
- Кнопка копирования для каждого сообщения
- Кнопка вставки текста из буфера обмена
- Автоматическое сохранение и продолжение сессий через CLI
- Переключатели песочницы (Sandbox) и авто-подтверждения действий
- Адаптивные цвета, подстраивающиеся под тему Noctalia

## Доступные команды в чате

| Команда | Описание |
|---------|----------|
| `/help` | Показать список доступных команд |
| `/clear` | Очистить локальную историю чата |
| `/new` | Начать новую сессию |
| `/stop` | Остановить генерацию ответа |
| `/copy` | Скопировать последнее сообщение ассистента |
| `/cwd <путь>` | Установить рабочую директорию |

## Управление через IPC (командную строку)

```sh
qs -c noctalia-shell ipc call plugin:agy-panel toggle
qs -c noctalia-shell ipc call plugin:agy-panel send "Hello"
qs -c noctalia-shell ipc call plugin:agy-panel newSession
```
